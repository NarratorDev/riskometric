# Riskometric Assessment Platform

## Behavioral Finance & Investor Psychology Assessment

Riskometric is an interactive behavioral finance assessment platform designed to demonstrate how psychological biases influence investment decision-making.

The project was developed for academic workshops and classroom activities at the Center of Investments and Capital Markets (CICM).

---

# Features

- Multi-question behavioral finance assessment
- Demonstrates Loss Aversion
- Tests Herd Mentality
- Tests Overconfidence Bias
- Tests Sunk Cost Fallacy
- Tests Certainty Effect
- Generates investor personality profiles
- Live facilitator dashboard
- Public web deployment using Render
- Responsive web interface

---

# Behavioral Biases Covered

## 1. Loss Aversion
People feel losses more intensely than equivalent gains.

## 2. Certainty Effect
People irrationally prefer guaranteed outcomes over probabilistic gains.

## 3. Sunk Cost Fallacy
Investors continue bad investments because of past commitments.

## 4. Herd Mentality
Investors follow crowd behavior during market hype.

## 5. Overconfidence Bias
Investors become excessively aggressive after short-term success.
